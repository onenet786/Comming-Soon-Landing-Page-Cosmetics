# Cosmetic ERP — Coming Soon Landing Page

A world-class "Coming Soon" landing page for Cosmetic ERP.  
Single-file, zero-build, zero-dependency. Works with any static file server.

## Stack
- Pure HTML5 / CSS3 / Vanilla JS
- Google Fonts (Inter) — loaded from CDN
- No npm, no build step, no framework

## Quick Preview
Open `index.html` directly in any browser.

## Deployment
See [DEPLOYMENT.md](./DEPLOYMENT.md) for step-by-step hosting instructions on a remote server running Node.js + aaPanel via `git clone`.
